# 📱 Phone Mouse — તમારા ફોનને લેપટોપના માઉસ તરીકે વાપરો

તમારા Android/iPhone ને Wi-Fi દ્વારા Windows લેપટોપના **વાયરલેસ ટચપેડ + કીબોર્ડ** તરીકે વાપરો. કોઈ એપ ડાઉનલોડ કરવાની જરૂર નથી — ફોનમાં ફક્ત બ્રાઉઝર ખોલવાનું છે.

## ✨ શું શું કરી શકાય
- 🖱️ **કર્સર ખસેડવું** — ફોન પર આંગળી ફેરવો
- 👆 **ડાબી ક્લિક** — ટેપ કરો (અથવા બટન દબાવો)
- 👉 **જમણી ક્લિક** — બટન દબાવો
- ⚡ **ડબલ ક્લિક** — "મધ્ય" બટન
- ↕️ **સ્ક્રોલ** — બે આંગળીથી ઉપર/નીચે ફેરવો (અથવા સ્ક્રોલ મોડ ચાલુ કરો)
- ✊ **ડ્રેગ** — ડબલ-ટેપ કરીને આંગળી દબાવી રાખીને ખસેડો
- ⌨️ **કીબોર્ડ** — ફોનથી ટાઇપ કરો, લેપટોપ પર દેખાશે
- 🎚️ **ગતિ (sensitivity)** સ્લાઇડરથી ઝડપ બદલો

## 🚀 કેવી રીતે ચલાવવું (Windows)

### પગલું 1: Node.js ઇન્સ્ટોલ કરો (ફક્ત પહેલી વાર)
જો તમારા લેપટોપ પર Node.js ન હોય તો https://nodejs.org પરથી ડાઉનલોડ કરીને ઇન્સ્ટોલ કરો.

### પગલું 2: સર્વર ચાલુ કરો
આ ફોલ્ડરમાં **`START-Windows.bat`** ફાઇલ પર **ડબલ-ક્લિક** કરો.

પહેલી વાર થોડી સેકન્ડ સેટઅપ માટે લાગશે. પછી સ્ક્રીન પર એક **QR કોડ** અને **સરનામું (address)** દેખાશે, જેમ કે:
```
http://192.168.1.5:3000
```

### પગલું 3: ફોન જોડો
1. ખાતરી કરો કે **ફોન અને લેપટોપ એક જ Wi-Fi** પર છે.
2. ફોનના કેમેરાથી **QR કોડ સ્કેન કરો**, અથવા ફોનના બ્રાઉઝરમાં ઉપરનું સરનામું ટાઇપ કરો.
3. બસ! હવે ફોન તમારું ટચપેડ બની ગયું છે. 🎉

## 🌐 બીજા નેટવર્કથી / ઇન્ટરનેટ દ્વારા વાપરવું (Netlify નહીં!)

> **મહત્વનું:** આ એપ **Netlify (કે કોઈ પણ ક્લાઉડ હોસ્ટ) પર ડિપ્લોય ન થઈ શકે.** કારણ: માઉસ તો *તમારા જ લેપટોપ* પર ખસવાનું છે, એટલે સર્વર પણ તમારા લેપટોપ પર જ ચાલવું જોઈએ. Netlify ફક્ત static પેજ આપે, સતત ચાલતું Node સર્વર કે WebSocket નહીં, અને એ ક્લાઉડ મશીન તમારા લેપટોપને કંટ્રોલ ન કરી શકે.
>
> તેના બદલે આપણે **Cloudflare Tunnel** વાપરીને તમારા લેપટોપના સર્વરને એક સુરક્ષિત જાહેર `https://` URL આપીશું. પછી ફોન ગમે ત્યાંથી (મોબાઇલ ડેટા પર પણ) જોડાઈ શકે.

### પગલાં
1. **cloudflared** ઇન્સ્ટોલ કરો (ફક્ત પહેલી વાર). PowerShell ખોલીને:
   ```powershell
   winget install --id Cloudflare.cloudflared
   ```
   (અથવા https://github.com/cloudflare/cloudflared/releases પરથી `.exe` ડાઉનલોડ કરો.)

2. **`START-Internet-Windows.bat`** પર ડબલ-ક્લિક કરો.
   - એક વિન્ડોમાં સર્વર ચાલશે અને **PIN** બતાવશે (દા.ત. `🔒 PIN: 4321`).
   - બીજી વિન્ડોમાં `https://something-random.trycloudflare.com` જેવું **જાહેર URL** આવશે.

3. ફોનના બ્રાઉઝરમાં તે `https://...trycloudflare.com` URL ખોલો (ફોન કોઈ પણ નેટવર્ક પર હોય તો ચાલે).

4. ફોન પર **PIN નાખો** → જોડાઈ જશો. 🎉

### 🔒 સુરક્ષા (ઇન્ટરનેટ વાપરો ત્યારે ખૂબ જરૂરી)
- સર્વર દર વખતે **રેન્ડમ 4-આંકડાનો PIN** બનાવે છે; સાચો PIN નાખ્યા વગર કોઈ કંટ્રોલ ન કરી શકે.
- 5 વાર ખોટો PIN નાખે તો કનેક્શન આપોઆપ બંધ થઈ જાય.
- પોતાનો ફિક્સ PIN જોઈતો હોય તો: ટર્મિનલમાં `set PMOUSE_PIN=1234` કરીને સર્વર ચલાવો.
- **કામ પૂરું થાય કે તરત બંને વિન્ડો બંધ કરો** — જેથી જાહેર URL બંધ થઈ જાય.
- ⚠️ trycloudflare નું URL જાહેર છે; **તે કોઈ સાથે શેર ન કરો.**

## 🔧 મુશ્કેલી આવે તો
- **પેજ ખુલતું નથી?** ફોન અને લેપટોપ એક જ Wi-Fi પર છે કે નહીં તે તપાસો.
- **હજુ નથી ચાલતું?** Windows Firewall ની પોપ-અપ આવે ત્યારે **"Allow / Permit"** પર ક્લિક કરો.
- **કર્સર નથી ખસતું પણ પેજ ખુલે છે?** સ્ક્રીન પર "ડેમો મોડ" લખ્યું હશે — એનો અર્થ `nut-js` ઇન્સ્ટોલ નથી થયું. `START-Windows.bat` ફરી ચલાવો, અથવા ટર્મિનલમાં `npm install` ચલાવો.
- **અલગ IP અજમાવો:** સર્વર ઘણા સરનામાં બતાવે તો બીજું પણ અજમાવો.
- **PIN ભૂલી ગયા?** સર્વર ચાલતી વિન્ડોમાં PIN લખેલો જ હોય છે (🔒 ની બાજુમાં).
- **ઇન્ટરનેટ મોડમાં "ખોટો PIN"?** સર્વર વિન્ડોમાં બતાવેલ તાજો PIN જ વાપરો (દર વખતે બદલાય છે).

## 🖥️ ટેકનિકલ માહિતી (વૈકલ્પિક)
- **Backend:** Node.js + Express + Socket.IO
- **માઉસ/કીબોર્ડ કંટ્રોલ:** `@nut-tree-fork/nut-js` (native)
- **Frontend:** સાદું HTML/CSS/JS — કોઈ ઇન્સ્ટોલ વગર ફોનના બ્રાઉઝરમાં ચાલે

મેન્યુઅલી ચલાવવા માટે:
```bash
npm install
npm start
```

> 🔒 **સલામતી:** આ ફક્ત તમારા લોકલ Wi-Fi નેટવર્ક પર ચાલે છે. એ જ નેટવર્ક પરનું કોઈ પણ ઉપકરણ આ સરનામું ખોલીને કંટ્રોલ કરી શકે, માટે ભરોસાપાત્ર (ઘર/ઓફિસ) નેટવર્ક પર જ વાપરો, જાહેર Wi-Fi પર નહીં.
